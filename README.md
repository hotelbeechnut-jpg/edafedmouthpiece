# Edafe Dmouthpiece — Personal Brand Website

Premium, professional multi-page website for **Edafe Omadevwe** (Edafe Dmouthpiece) — Nigerian OAP, MC, comedian, actor, football analyst, and content creator.

## Tech Stack

- **Framework:** Next.js (Pages Router)
- **Styling:** CSS Modules + CSS Custom Properties
- **Fonts:** Playfair Display (headings), Inter (body)
- **Build:** Turbopack

## Pages

1. **Home** — Hero section with editorial portrait, role highlights, CTA to booking
2. **About** — Full bio, brand story (placeholder), photo gallery strip
3. **Portfolio** — Segmented by Comedy/MC, Hosting/Events, Sports Analysis, with flyer proofs
4. **Booking** — Inquiry form with validation, service info, contact shortcuts
5. **Testimonials** — Placeholder testimonial cards ready for real quotes
6. **Contact** — Contact form, social links, location info

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
